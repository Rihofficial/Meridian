**🏡 Meridian — Luxury Real Estate Website

A modern, responsive real estate landing page template designed to showcase premium properties and provide an elegant user experience for property search and browsing.

This project features beautifully styled UI components like featured listings, property search filters, statistics sections, and agent profiles — perfect for real estate agencies, independent brokers, or portfolio showcases.

**📦 Features

✔ Fully responsive modern UI built with HTML, CSS & JavaScript
✔ Hero section with call‑to‑action links
✔ Property search bar with filters like location, type, price and bedrooms
✔ Featured properties grid
✔ Agent profile section
✔ Listings with interactive cards
✔ Smooth scrolling and reveal animations
✔ Clean, reusable styles

🛠 How To Use
✅ 1. Clone the repository
git clone https://github.com/Rihofficial/Meridian.git
✅ 2. Navigate into the project
cd Meridian
✅ 3. Open in browser

Double‑click index.html or serve it with a simple dev server:

npx serve .
💡 Customization Points

You can personalize the template for your real estate business:

✨ Update property images & details
✨ Change the hero headline & tagline
✨ Replace agent info and contact links
✨ Adjust price ranges & filters
✨ Add back‑end search integration (optional)

📌 Scripts

The script.js file handles interactive elements like:

✔ Property cards reveal animation
✔ Custom cursor animations
✔ Mobile navigation toggle
✔ Interactive stats counters

You can expand it to include more UI behavior or connect APIs.

No external dependencies — this is a no‑build, pure web frontend.

You can integrate frameworks or build tools like React, Tailwind, or Webpack if planning to scale this further.
